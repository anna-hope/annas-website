## Build Setup

### install dependencies
- Zola: https://github.com/getzola/zola/releases/tag/v0.18.0

Then

```bash
yarn install
```

# serve with hot reload at localhost:1111
```bash
zola serve
```

# build for production 
See `build` section in `netlify.toml`
